## Reporting a Vulnerability

**Note:** As this project is no longer maintained, security vulnerabilities will not be actively addressed.

However, if you discover a critical security issue, you may still report it for documentation purposes:

1. **Contact**: Open a [GitHub Issue](https://github.com/Stensel8/Half-a-Minute-IT-Madness-Android/issues) with details.
2. **Describe**: Clearly describe the vulnerability, including steps to reproduce it.
3. **No Response Guarantee**: Due to the End of Life status, we cannot guarantee any response or fix.

This project was a school assignment and is provided as-is for archival purposes only.
